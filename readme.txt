Manage custom media entries from the WordPress media library with custom names, paths, and extension settings.

== Description ==

WP Media Manager lets you create a curated list of media entries from your WordPress media library, giving each file a custom name, custom path prefix, and control over whether the extension is included in the output string.

**Key Features:**

* Browse and select media via the native WordPress Media Library modal
* Paste any WordPress uploads URL to auto-detect the attachment
* Set a custom name and custom path for each entry
* Toggle file extension inclusion per entry
* Live output preview before saving
* Manage all entries in a clean, responsive card grid
* AJAX-powered save, edit, delete, and search — no page reloads
* Image preview, PDF iframe preview, video player preview, audio player
* Toast notifications for all operations
* Translation-ready (`.pot` file included)

== Installation ==

1. Upload the `wp-media-manager` folder to `/wp-content/plugins/`.
2. Activate the plugin through **Plugins → Installed Plugins**.
3. Navigate to **Media Manager** in the WordPress admin sidebar.

== Frequently Asked Questions ==

= Does deleting an entry remove the actual file? =
No. Deleting an entry only removes the plugin's database record. Your media file stays intact in the WordPress Media Library.

= Which file types are supported? =
Images, PDFs, videos, audio files, Word documents, spreadsheets, and any other file type you have in your media library.
